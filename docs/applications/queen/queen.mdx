# Queen

Queen is part of a micro frontend application called Sabiane. In Sabiane there are 2 JavaScript front-end, Pearl Jam and Queen.

- [Pearl Jam](https://github.com/InseeFr/Pearl-Jam/) is responsible for the case management for Computer-Assisted Personal Interviewing (CAPI).
- [Queen](https://github.com/InseeFr/Queen) is responible for displaying surveys for Computer-Assisted Personal Interviewing.

Pearl Jam is the management application and the container of the microfronted.

Queen is integrated in Pearl through a web component [here](https://github.com/InseeFr/Pearl-Jam/blob/8ef6f31e5f98c6822fb01d8a020db279fc2fdf89/src/components/panel-body/queen-container/component.js)

# Code Repository

All the software are open source and available on Github of the organization [InseeFr](https://github.com/inseeFr/). There are both under [MIT Licence](https://github.com/InseeFr/Queen/blob/781595774fa44a738812b6ac25e3af3aa0a2e40d/LICENSE).

## Back-end

Queen communicates with a Spring boot API. This API in maintained inside the Collection platform product. You can browse the code repository [here](https://github.com/InseeFr/Queen-Back-Office/)
